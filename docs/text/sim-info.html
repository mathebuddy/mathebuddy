This page enables to simulate courses. Since the simulator is written with Flutter, the behavior is identical to the app.

Explore the demo course

Remark: The demo course is currently not editable.

The demo course demonstrates how to write courses in the mathe:buddy langauge (MBL). Each file concentrates on a separate technological aspect.

Steps:
  1. Click on the [DEMO] button above.
  2. Choose file HELLO.MBL. You will see the MBL code on the right info page.
  3. Optionally click on [MBCL] to see the compiled code, that is transferred to the simulator.
  4. Check in the log area (below the simulator window) that no error occurred.
  5. Start the simulator by pressing on the power on button . The current course (or level) is simulated.
  6. Optionally change to another screen size by pressing on one of the smartphone buttons .

Create own courses

Remark: The following guide will currently not work with all browsers. As a workaround, you may run this website locally on your computer, as described here in the readme file.

Creating and editing own courses requires to store MBL files locally on your device. As a dependency you need to have Python 3 installed on your computer.

Steps:
  1. Download an extract the example template from here. Alternatively, run the following command from a terminal window: git clone https://github.com/mathebuddy/mathebuddy-public-courses.git
  2. Change into the mathebuddy-public-courses directory and run server.py. The script runs a local webserver at port 8271.
  3. On this website, click on the [HTTP://LOCALHOST:8271] button above.
  4. The remaining steps are similar to working with the DEMO course (above).
